# Промпт: Тестирование модели из библиотеки Pulsar

## Задача
Проверить работу существующей модели из моей библиотеки и узнать её реальные параметры в симуляции.

## Компонент для теста
[ВСТАВЬ НАЗВАНИЕ И ТИП, например: LM741 (ОУ) или 1N4007 (Диод)]

## Что нужно сделать

### 1. Прочитай модель
Используй read чтобы открыть файл из нужной папки:
- ОУ: ~/Pulsar/Mod/opamp/
- Диод: ~/Pulsar/Mod/Diodes/
- BJT: ~/Pulsar/Mod/BJT/

### 2. Создай тестовую схему (Testbench)
Напиши SPICE-нетлист для проверки компонента и сохрани его в папку tests через write:
- Для ОУ: инвертирующий усилитель (Ku=10), AC и Transient анализ. Файл: ~/Pulsar/tests/test_OPAMP.cir
- Для Диода: DC sweep от 0 до 1.5В для снятия ВАХ. Файл: ~/Pulsar/tests/test_DIODE.cir
- Для BJT: DC sweep для снятия выходной характеристики. Файл: ~/Pulsar/tests/test_BJT.cir

### 3. Запусти симуляцию напрямую через ngspice
Используй bash для запуска команды в терминале (это быстрее, чем через MCP):
ngspice -b ~/Pulsar/tests/test_[ИМЯ].cir > ~/Pulsar/tests/result_[ИМЯ].txt

### 4. Проанализируй результаты
Прочитай текстовый файл с результатами (read) и найди ключевые параметры:
- ОУ: реальная полоса пропускания и скорость нарастания (slew rate).
- Диод: падение напряжения (Vf) при токе 10мА.
- BJT: коэффициент усиления (hFE) в рабочей точке.

### 5. Создай отчёт
Напиши краткий отчёт и сохрани его в ~/Pulsar/tests/report_[ИМЯ].md. 
Укажи: какие параметры подтвердились, какие отклонения есть, и подходит ли эта модель для точных расчётов в Pulsar.
