# Промпт: Создание модели MOSFET для Pulsar

## Контекст
Я использую программу Pulsar (редактор схем + ngspice симулятор). 
Мне нужна SPICE-модель MOSFET-транзистора для добавления в библиотеку.

## Задача
Создай файл .lib в формате Berkeley SPICE на основе параметров из datasheet.

## Название MOSFET
ME4856

## Что нужно сделать

### 1. Загрузи skill
Загрузи skill из файла `~/Pulsar/skills/pulsar-modeling.md`

### 2. Найди параметры из datasheet
Используй `websearch` для поиска datasheet и извлеки:
- GBW (полоса единичного усиления)
- Slew rate (скорость нарастания)
- Входное сопротивление
- Выходное сопротивление
- Напряжение насыщения
- Тип входов (JFET, BJT, CMOS)

### 3. Создай модель
Используй `spicebridge_create_model` для создания макромодели.
Модель должна иметь 5 выводов: in+, in-, VCC+, VCC-, OUT

### 4. Протестируй модель
Собери тестовую схему (инвертирующий усилитель Ku=10):
- R1 = 1k (входной резистор)
- R2 = 10k (резистор обратной связи)
- Питание ±15В
- Вход: синус 1 кГц, амплитуда 100 мВ

Запусти:
- `spicebridge_run_ac` от 1 Гц до 10 МГц (проверка полосы)
- `spicebridge_run_transient` на 10 мс (проверка slew rate)

### 5. Проверь результаты
- Полоса пропускания должна быть ≈ GBW/Ku = GBW/10
- Slew rate не должен превышать значение из datasheet
- Модель должна сходиться без ошибок

### 6. Запиши в библиотеку Pulsar
Используй `write` для создания файла:
`~/Pulsar/Mod/opamp/[НАЗВАНИЕ_ОУ].lib`

Файл должен содержать:
- Комментарий с названием и параметрами
- Блок .SUBCKT с 5 выводами
- Код модели
- Блок .ENDS

### 7. Обнови README
Добавь информацию о новой модели в `~/Pulsar/Mod/README.md`:
- Название
- Производитель
- Ключевые параметры (GBW, SR, тип входа)
- Дата создания
